可变数据类型和函数

在数据类型和运算符课程中，当我们在列表和成员运算符部分介绍列表时，讲解了数据类型是可变类型或不可变类型的含义。此外，我们在函数课程的变量作用域部分介绍了不可变 python 数据类型的作用域。

在此视频中，我们将复习可变和不可变数据类型的定义，并详细讲解这两个类别分别包含哪些数据类型。此外，我们将介绍对于函数来说，可变数据类型的行为如何。这部分将帮助你更好地理解如何在函数中使用可变数据结构（例如列表和字典）。这样有助于理解如何在 check_images python 程序中使用各种数据结构。
