17. 23. 样本和总体

  1. 样本和总体

Start Quiz:

INSTRUCTOR NOTE:

使用随机变量（n = 100），研究者发现一般美国居民每周的上网时间为 32 小时。假设美国居民整个总体的上网时间真值为 25 小时/周。在此情境下，以下哪个说法是正确的？选择所有适用项。

  • □ 样本统计量为 32，总体参数为 25。
  • □ 32 和 25 之间的差异称为“抽样误差”。
  • □ 样本值是错误的，因为样本是随机的。
  • □ 样本均值和总体均值之间存在差异是正常的。
  • □ 样本统计量为 25，总体参数为 32。
  • □ 此示例中的抽样误差量为 7 小时/周（32 - 25）。
  • □ x̅ = 32；μ = 25
  • □ μ = 32； x̅ = 25
  • □ 样本中包含 100 名美国居民。
  • □ 如果我们使用更大的样本（例如，n = 1000），获得的样本均值往往更接近总体均值。
  • □ 我们的样本均值和总体均值不同是不正常的，因为随机样本应保证 100% 的准确估计。
