该编写代码了！在此练习中，你将实现感知器算法以分类下面的数据（位于文件 data.csv 中）。

感知器步骤如下所示。对于坐标轴为
(p,q)
的点，标签 y，以及等式
\hat{y} = step(w_1x_1 + w_2x_2 + b) 给出的预测

  • 如果点分类正确，则什么也不做。
  • 如果点分类为正，但是标签为负，则分别减去
    \alpha p, \alpha q,
    和
    \alpha
    至
    w_1, w_2,
    和
    b
  • 如果点分类为负，但是标签为正，则分别将
    \alpha p, \alpha q,
    和
    \alpha
    加到
    w_1, w_2,
    和
    b
    上。

def perceptronStep(X, y, W, b, learn_rate = 0.01):
    for i in range(len(X)):
        y_hat = prediction(X[i],W,b)
        if y[i]-y_hat == 1:
            W[0] += X[i][0]*learn_rate
            W[1] += X[i][1]*learn_rate
            b += learn_rate
        elif y[i]-y_hat == -1:
            W[0] -= X[i][0]*learn_rate
            W[1] -= X[i][1]*learn_rate
            b -= learn_rate
    return W, b
