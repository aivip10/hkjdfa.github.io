18. 39. SAD

  1. SAD

Question:

Start Quiz:

INSTRUCTOR NOTE:

一些小型研究发现模拟曙光可以有效治疗季节性情绪失调症 (SAD)。近期的一项研究将此治疗方法与亮光疗法和安慰剂控制条件（Avery 等人，2001 年）做了比较。患有 SAD 的 95 个 SAD 患者被随机分到了以下三种条件中的一个：

  1. 亮光疗法（10,000勒克斯 – 国际标准照明单位，持续 30 分钟，从上午 6:00 至 6:30），人数 n = 33
  2. 曙光模拟（凌晨 4:30 至 6:00 一个半小时的曙光信号治疗，峰值 250勒克斯），人数 n = 31
  3. 安慰剂条件：暗红光（从凌晨 4:30 到 6:00 一个半小时的曙光信号治疗，峰值 0.5勒克斯），人数 n = 31

在之后的 6 周时间内，由一名精神病医生使用汉密尔顿抑郁量表结构性面试指南-季节性情绪失调症版本 (SIGH - SAD) 对这些研究对象进行盲评，可测量出研究对象的此种抑郁症的水平，分数在 0 到 89 之间，分数越高表明情况越严重。研究者报告相比安慰剂和亮光疗法，曙光模拟法的康复率更高。
（来源：http://dx.doi.org/10.1016/S0006-3223(01)01200-8）

此研究中的自变量是什么？

  • □ SAD 的严重程度
  • □ 光疗法的类型
  • □ 患者在 SIGH-SAD 中的得分。
  • □ 以勒克斯为单位的光的强度
