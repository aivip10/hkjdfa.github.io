04. 截止时间政策

截止时间政策

本课程以项目为基础，每个项目都有相应的建议截止时间。我们给你提供了较为灵活的项目完成时间，只要在课程有效期内均可提交。不过为了保证满足毕业要求，我们强烈建议你按时完成项目。

如果我没有按时完成项目怎么办？

要想成功毕业，你必须在开始上课的16周内完成和提交项目，并达到项目要求。虽然错过截止日期并不会受到惩罚，但在错过一个截止日期后，倘若你没能保持进度，在学期结束前没能完成所有要求的项目，你将可能被取消学习资格 。最后，推荐你可以从论坛和微信群中获取社群助教和导师对你的帮助，与你的同学共同进步。

如果我没有在学期结束前完成课程学习会怎么样？

如果你没能在规定的16周内完成本学期的学习，你将被取消学习资格，并无法学习课程内容。为了恢复课程学习，你需要再次缴纳学费。课程恢复后，你的进度将被保留，你可以从停滞的部分继续学习。
