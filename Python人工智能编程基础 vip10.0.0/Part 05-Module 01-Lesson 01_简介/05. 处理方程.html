05. 处理方程

处理方程

优达学城将通过检查你得出的数学计算结果，快速有效地检测你的数学知识！

在每个部分结束时，你都要完成一道练习。在练习中，我们有时候会将你得出的方程与我们的方程进行比较。我们还可能会检查得出的数字值。

你需要在编辑窗口的 math.txt 下写出你的方程。系统会将你的方程与正确方程进行比较，立即告诉你结果正确与否。

我们来看一个示例：

在此示例中，将检查两个正确的方程：

  • \sqrt{x^e + y^9} = z^3
  • (x+1) * (x-1) = y

正确答案以绿色突出显示。

错误答案将以红色突出显示。

注意，我们测试的方程是 \sqrt{x^e + y^9} = z^3，但是在 math.txt 下（你需要在此处输入你的方程），我们使用了 sqrt{x^e + y^9} = z^3（写成 LaTeX 形式，这是我们用作数学编写工具的语言）。

要使用 Workspace 并测试你的方程，你需要以 LaTeX 格式输入答案。你会发现该工具操作简便，不需要掌握很多经验知识。

  • 要了解如何使用 LaTeX 的更多示例，请访问下面这两个链接：

    • Latex cheatsheet

    • LaTeX Cookbook

最后，你可能需要重置代码，这一点也十分重要。

打开窗口左下角的菜单栏并选择重置代码。

并在提示符处输入 Reset data（请注意区分大小写）。

有时候，当你尝试打开 Workspace 时，可能会见到以下画面：

如果发生这种情况，请等几秒钟，然后重新打开窗口。

成功了！现在可以测试 Workspace 了！
