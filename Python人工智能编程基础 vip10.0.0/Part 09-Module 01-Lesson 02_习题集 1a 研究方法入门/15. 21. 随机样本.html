  1. 随机样本

Start Quiz:

INSTRUCTOR NOTE:

随机样本是你可以：

  • □ 根据从样本那里获取数据的容易度选择个体。
  • □ 根据其是否愿意回答你的问题选择个体。
  • □ 以选中概率相同的方式选择个体。
  • □ 以选中一个个体不会影响另一个个体被选中几率的方式选择个体。
  • □ 选择你的朋友、家人和邻居参与你的调查。
