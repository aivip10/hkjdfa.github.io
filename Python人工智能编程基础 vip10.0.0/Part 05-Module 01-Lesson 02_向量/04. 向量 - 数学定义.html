04. 向量 - 数学定义

向量 - 数学定义

什么是向量？简单解释下，向量是有序的数字列表。

向量中的每个元素（亦称为组件或坐标）是一个数字，此处用 a_i 表示。

这个向量（上图中）有 n 个元素，可以位于向量空间或是实数 \mathbb{R} 中。

具有 n 个真实元素的向量定义了 n 维向量，属于向量空间 \mathbb{R}^n。

我们使用以下数学记法定义向量：\vec{x}

在上述示例中：

  • 向量 \vec{x} 位于向量空间 \mathbb{R}^n 中。或者用数学记法表示为：\vec{x}\in\mathbb{R}^n

  • \vec{x}=\begin{bmatrix} x_1\\ x_2\\ x_3\\ :\\x_n\end{bmatrix}

和视频中一样，我们以更直观的方式讲解，并重点研究实数域2D向量(即二维平面向量)。换句话说，我们将重点查看 \mathbb{R}^2 中的向量，该向量定义了平面上的所有点。

我们来看看下图：

要从点 B 到点 A，我们需要向右走 4 步，再向上走 2 步。

这样就定义了向量 \vec{x}=\begin{bmatrix} 4\\2\end{bmatrix}。
