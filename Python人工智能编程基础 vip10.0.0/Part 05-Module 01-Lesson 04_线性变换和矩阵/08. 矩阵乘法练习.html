08. 矩阵乘法练习

矩阵乘法练习

假设 A 是一个 1 x 5 矩阵，B 是一个 5 x 3 矩阵。

A=\begin{bmatrix} 0.6 &-15 &2&5&98\end{bmatrix}

B=\begin{bmatrix} 2&2&-4\\ 9 &-14 &0\\13&-0.5&44\\1&9&4\\0&0&5\end{bmatrix}

注意 A 实际上是一个行向量！只有一行或一列的矩阵是一个向量

  • 只有一行的矩阵是行向量
  • 只有一列的矩阵是列向量

如果 C=AxB

(a) C 将有多少行？

(b) C 将有多少列？

(c) c_{13} 的值将是多少？

(d) 如果 C=BxA，c_{13} 的值将是多少？
