06. 方形矩阵乘法练习

方形矩阵乘法练习

假设 A 和 B 是两个 3 x 3 方形矩阵。

A=\begin{bmatrix} 3 &1 &2\\ -5 &4 &1\\0&3&-8\end{bmatrix}

B=\begin{bmatrix} 0 &5&-1\\ 3 &2 &-1\\10&0.5&4\end{bmatrix}

(a) 矩阵 C 的元素 c_{23} 的值是多少，其中 C=A x B

(b) 矩阵 C 的元素 c_{23} 的值是多少，其中 C=B x A

注意 A x B\neqB x A，因为
A x B 中的元素 c_{23} 与 B x A 中的元素 c_{23} 不同。

因为 A x B\neqB x A，我们称之为不满足交换律。这是一个有趣的现象，标量乘法肯定是满足交换律的。
