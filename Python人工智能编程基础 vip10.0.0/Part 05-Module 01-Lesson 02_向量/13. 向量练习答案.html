13. 向量练习答案

练习 1 的答案

向量的大小是：

\sqrt{a^2+b^2+c^2} 。

在 workspace 中需要使用以下格式：\sqrt{a^2+b^2+c^2}。

练习 2 的答案

算出加法结果后，可视化向量。绘出图形后，你将发现很容易就能找到答案。

  • 大小：1
  • 方向：0{^\circ}（请在 workspace 填写 0）

练习 3 的答案

(a) 大小现在是：

|\alpha|\sqrt{a^2 + b^2}.

在 workspace 中需要使用以下格式：\alpha\sqrt{a^2 + b^2}。

(b) 方向会改变吗？

不会

(c) 方向将改变 180{^\circ}

（仅在 workspace 填写 180）

(d) 大小将

减小
